# Scheduled Tasks — рекомендуемый набор

> Регистрируются через Cowork → Scheduled (или через MCP `mcp__scheduled-tasks__create_scheduled_task`).
> Работают **только при запущенном Claude Desktop**. Для критичного мониторинга — GitHub Actions (отдельный файл).

---

## Рекомендуемый набор для соло-разработчика

### 1. Утренний дайджест (рабочие дни)

- **Cron**: `0 9 * * 1-5` (09:00 пн-пт)
- **Промпт**:
  > Сделай утренний дайджест:
  > 1. Открытые PR во всех моих репозиториях (GitHub MCP)
  > 2. Упавшие CI за последние 24 часа
  > 3. Новые issues, отмеченные `priority:high`
  > 4. Напоминания из TODO.md проектов
  >
  > Формат: 5 пунктов, не более 15 строк. Если ничего нет — короткое «всё спокойно».
- **Канал**: Cowork chat (новая task создаётся автоматически)

### 2. Проверка CI каждые 30 минут (рабочее время)

- **Cron**: `*/30 9-19 * * 1-5`
- **Промпт**:
  > Опроси GitHub Actions по моим репозиториям за последние 30 минут. Если есть упавший workflow — сразу пуш в ntfy.sh с приоритетом high, текстом «🔴 CI упал в <repo>/<workflow>, ссылка: <URL>». Если всё зелёное — молчи.
- **Канал**: ntfy.sh (на телефон) + Cowork (если открыт)

### 3. Пятничный tech-debt дайджест

- **Cron**: `0 17 * * 5` (пятница 17:00)
- **Промпт**:
  > По каждому активному репозиторию запусти skill `engineering:tech-debt`. Сводный отчёт: топ-5 проблем по всем проектам с оценкой effort/impact. Сохрани в Cowork artifact `tech-debt-<дата>`.
- **Канал**: Cowork artifact + Cowork chat сводка

### 4. Еженедельная резервная копия

- **Cron**: `0 22 * * 0` (воскресенье 22:00)
- **Промпт**:
  > Запусти scripts/backup-321.ps1, проверь что бэкап успешен. Если ошибка — пуш в ntfy.sh.

### 5. Ежемесячная самооптимизация шаблона

- **Cron**: `0 10 1 * *` (1-е число месяца, 10:00)
- **Промпт**:
  > Прочитай все LAB_JOURNAL.md за последний месяц. Найди:
  > - Повторяющиеся проблемы
  > - Часто блокировавшиеся hooks
  > - Низкие Confidence (< 70%)
  > Предложи 3 улучшения для project-template/. Сохрани в Cowork artifact `template-evolution-<месяц>`.
- **Канал**: Cowork artifact (читать на досуге)

### 6. Конкурентная разведка (опционально)

- **Cron**: `0 11 * * 1` (понедельник 11:00)
- **Промпт**:
  > Поищи на GitHub новые репозитории по темам моих проектов (web-app, trading, 44-FZ) с фильтром: star > 100, created < 30 дней. Дайджест в Cowork artifact `competitor-watch-<дата>`.

---

## Как зарегистрировать через UI

1. Cowork → Scheduled
2. New scheduled task
3. Заполнить: имя, cron, промпт, канал доставки
4. Save

## Как зарегистрировать пакетно через MCP

Можно попросить AI: «Зарегистрируй все scheduled-задачи из cowork-config/scheduled-tasks.md». AI вызовет `mcp__scheduled-tasks__create_scheduled_task` по списку.

## Ограничения

- Задачи **не работают**, когда Claude Desktop закрыт
- Минимальный интервал — 1 минута (не реальное real-time)
- Для критичного мониторинга 24/7 — использовать GitHub Actions cron (запускается в облаке GitHub)
